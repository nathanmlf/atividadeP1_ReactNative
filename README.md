# Atividade P1 - React Native
## Aluno: Nathan Mendonça Lourenço Ferreira

O projeto é uma tentativa de clonar o iniciar do aplicativo "Uber" e a página principal do aplicativo, a falta dos ícones dificultou a estilização...

### Como acessar as duas páginas do projeto

A página do iniciar é o App1, a página principal é o App2, basta mudar a numeração e rodar o projeto!
